# 【山田さん専用】歯科クリニックチェックツール セットアップガイド

このツールを山田さんのPCで直接動かせるようにするための手順です。
一度設定すれば、田中さんのPCの状態に関わらずいつでもツールを使用できるようになります。

---

## ステップ1：Python（パイソン）のインストール

ツールを動かすための「エンジン」をインストールします。

1. [Python公式サイト](https://www.python.org/downloads/windows/)を開きます。
2. **「Download Python 3.12.x」**（一番上の黄色いボタン）をクリックしてインストーラーを保存します。
3. 保存したファイルを実行します。
4. **【重要】** インストール画面の下部にある **「Add Python.exe to PATH」に必ずチェックを入れてから** 「Install Now」をクリックしてください。
    - ※これにチェックを入れないと、コマンドが動かなくなります。
5. 「Setup was successful」と出たら「Close」で閉じます。

---

## ステップ2：プログラムのコピー

1. 田中さんから共有された `dental-checker` フォルダ（一式）を、山田さんのPCの好きな場所（デスクトップなど）に保存してください。
2. フォルダを開き、中身が表示されることを確認します。

---

## ステップ3：準備（初回のみ）

1. フォルダ内の何もない場所で「右クリック」→ **「ターミナルで起動」**（または「PowerShellで開く」）を選択します。
2. 黒い（または青い）画面が開くので、以下の文字をコピーして貼り付け、Enterキーを押してください。
   ```bash
   pip install -r requirements.txt
   ```
3. 数分待ち、エラーが出ずに終われば準備完了です。

---

## ステップ4：アプリの起動（使うときは毎回これだけ）

1. `ステップ3` と同様にフォルダ内でターミナル（黒い画面）を開きます。
2. 以下の文字を入力して、Enterキーを押してください。
   ```bash
   streamlit run app.py
   ```
3. 自動的にブラウザが立ち上がり、アプリが表示されます！

---

### うまくいかないときは？
- 「コマンドが見つかりません」と出る場合：ステップ1の「Add Python.exe to PATH」へのチェックが漏れている可能性があります。Pythonを一度アンインストールして、チェックを入れて入れ直してください。
- 画面がすぐ消える場合：田中さんに `.env` ファイルや `APIパスワード` の設定状況を確認してください。
